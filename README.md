# Prototype Beeld in PGO

## Doel

Maak een visueel prototype om schermontwerp, informatiestructuur en UX te valideren.
Geen echte authenticatie, geen live data-koppelingen.

## Wat is een PGO?

Een persoonlijke gezondheidsomgeving (PGO) is een app of website waarin je informatie over je eigen gezondheid bij kunt houden en waarmee je actief aan de slag kunt gaan met je gezondheid. Zo kun je jouw medische gegevens verzamelen en beheren. Op deze manier houd je grip op je gegevens: van behandelingen tot labuitslagen, medicijngebruik en inentingen.
https://medmij.nl/wat-is-een-pgo/
